How to push code to github

First of all we must to have a github acount than we should to config our name and email

Step1: Making local folder

We have created a local folder on our computer,

Step2: github repository

In this step we have to open github and creat a repository

Step3: Terminal

Open the terminal, here we are going to set our local project as the current directory,
than we will tiype ls, this will list the files that are inside of this folder

Step4: adding the files in our local folder

here we are going to add the files in our local folder to like git repository,
we are staging our files for their first commit,
we type git add period this will add all of the files inside of that folder.

Step5: git status

we are going to type git status and hit return, we will now see all of the files
that we have staged for commit.

Step6: git commit

In this step we are going to commit the files from our local repository,
we type: 

git commit -m ""
and inside of quotation marks we type first commit

Step7: git remote

In this step we need to specify the remote repository URL so we come back to our github repository and we copy the URL

step8: URL

We'll add the URL for the remote repository we do this by typing the following; 

git remote add origin URL
and we paste the URL

Step9: git push

Now we can push the code in our local repository to github,
we type the following; 

 git push --set-upstream origin
        master
or git push -you origin master we hit return,
now we will be asked to enter our username and password, once we've completed all of these steps
we can come back to github and we can view our files inside of our repository,
